🧮 3. The Step-by-Step Calculation

To find the value for the Resultant Matrix at index [0, 0], we multiply Row 0 of A by Column 0 of B, element by element, and add them together.

Calculating Row 0 of Resultant Matrix

Index [0,0]: (1×1) + (5×0) + (6×1) = 1 + 0 + 6 7
Index [0,1]: (1×2) + (5×1) + (6×0) = 2 + 5 + 0 7

⚠️ 5. The Golden Rules for Programming

You cannot just multiply any two random matrices. Before your loops even run, your program must check if multiplication is mathematically possible.

If Matrix A is m × n (rows × cols) and Matrix B is p × q (rows × cols):

RULE 1: The Multiplication Check

You can only multiply if the Columns of A equal the Rows of B.

if (n == p)

(If not equal, your code must print an error and stop).

RULE 2: The Size of the Result

If they can be multiplied, the resulting Matrix C will always have the dimensions:

m × q

(Rows of A × Columns of B).

🧠 6. Test Your Logic (Homework)

Apply the Golden Rules to answer the questions the instructor left on the board:

Question 1:

Matrix A is 3 × 4. Matrix B is 3 × 2.

Can they be multiplied? If yes, what is the size of Resultant Matrix C?

Answer: ❌ No, they cannot be multiplied. The columns of A (4) do not equal the rows of B (3).
Question 2:

Matrix A is 2 × 3. Matrix B is 3 × 2.

Can they be multiplied? If yes, what is the size of Resultant Matrix C?

Answer: ✅ Yes, they can. Columns of A (3) equals rows of B (3). By Rule 2, the Resultant Matrix C will be 2 × 2.
