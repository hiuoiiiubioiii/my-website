C Programming: Writing to Files

In-depth study notes based on Jenny's Lectures

1. The Core Concept: Hard Disk vs. RAM

When you run a C program, it executes inside the RAM (Main Memory) because it is fast. However, RAM is volatile. To save data permanently, we use files stored on the Hard Disk.

We cannot write directly from our C program to the Hard Disk. Instead, a copy (or buffer) is created in the RAM. We write to this buffer, and the system handles transferring that data to the actual file on the Hard Disk.

The Two Scenarios of Writing

  • Scenario 1: File Already Exists: A copy of the file is loaded into the RAM buffer. You write to the buffer, and it updates the Hard Disk.
  • Scenario 2: File Does NOT Exist: The system first creates a brand new file on the Hard Disk, then opens it in the RAM buffer so you can write to it.

2. Setting Up: File Pointers & fopen()

To interact with a file, you must declare a pointer of type FILE. This pointer holds the address of the file object loaded into memory.

#include <stdio.h>

void main() {
    FILE *fp = NULL; // Best practice: Initialize to NULL
    
    // Open the file
    fp = fopen("abc.txt", "w");
}

The fopen function takes two arguments:

  • File Name/Path: "abc.txt". If no path is provided, it assumes the file is in the same directory as your .c file.
  • Mode: "w" (Write Mode).

🚨 The Danger of "w" (Write) Mode

CRITICAL BEHAVIOR:
When you open an existing file in "w" mode, C instantly erases (overwrites) all previous content in that file. It opens it completely blank. If you want to keep the old data and add to the end, you must use Append mode ("a"), not Write mode.

3. The Safety Check (Error Handling)

Sometimes, a file cannot be opened (e.g., insufficient permissions, disk full). If fopen fails, it returns NULL. Always check for this before writing!

Note: You must include <stdlib.h> to use the exit() function.

#include <stdlib.h>

if (fp == NULL) {
    printf("Error: File could not be opened.\n");
    exit(1); // Terminates the program
}

4. Writing Data Function 1: fputc

fputc stands for "file put character". It is used to write exactly one single character into the file.

char ch = 'a';
fputc(ch, fp); // Writes the letter 'a' to the file

Pro Tip: Looping fputc for Strings

Though fputc writes one character, you can use a loop to write a whole string, character by character. (Requires <string.h> for strlen).

char str[50] = "Jenny Khatri";

for (int i = 0; i < strlen(str); i++) {
    fputc(str[i], fp);
}

5. Writing Data Function 2: fputs

If you don't want to use a loop, you can use fputs ("file put string"). This function writes an entire string (character array) to the file at once.

char str[50] = "Jenny's Lectures";

// Syntax: fputs(string_variable, file_pointer);
fputs(str, fp);

// OR pass the string directly:
fputs("Jenny Khatri", fp);

In her live coding, Jenny paired this with gets(str) to take a string from the user at runtime and write it directly to the file!

6. Writing Data Function 3: fprintf

What if you want to write a mix of numbers, strings, and floats? You use Formatted Output via fprintf.

It works exactly like the standard printf you already know, but you pass the file pointer (fp) as the very first argument.

int a = 10;
char str[50] = "Jenny Khatri";

// Syntax: fprintf(pointer, "format specifiers", variables);
fprintf(fp, "%d %s", a, str);

This will write the exact text "10 Jenny Khatri" into abc.txt.

7. The Final Step: fclose()

You must always close your file when you are done. Why?

  • It frees up the RAM being used by the file buffer.
  • It acts as a "Save" button—ensuring all the data lingering in the RAM buffer is officially flushed and written to the Hard Disk.
fclose(fp);

💻 Complete Live Coding Example

Here is the final, complete code putting all the pieces together (similar to her VS Code demonstration):

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

void main() {
    FILE *fp = NULL;
    char str[50];
    int age = 25;

    // 1. Open File
    fp = fopen("abc.txt", "w");

    // 2. Safety Check
    if (fp == NULL) {
        printf("Error opening file!");
        exit(1);
    }

    // 3. Get input and Write (Using fprintf for mixed data)
    printf("Enter your name: ");
    gets(str);
    
    fprintf(fp, "Name: %s, Age: %d", str, age);

    // 4. Close File
    fclose(fp);
    
    printf("Data successfully written to abc.txt");
}
