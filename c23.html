scanf() In-Depth

Formatted Input & Memory Management in C

1. Introduction: What is scanf()?

Definition: scanf() is the built-in "formatted input function" in C.

Purpose: It allows your program to read data entered by the user via the keyboard (runtime input) and store that data into variables.

Library: It lives in the Standard Input/Output library. You must tell the compiler to load it.

#include <stdio.h>

int main() {
    // scanf is now ready to use!
    return 0;
}

2. Static vs. Dynamic Input

Why use scanf? If we hardcode values (Static), our program does the exact same thing every time it runs. Dynamic input makes programs interactive.

Static (Hardcoded):

int a = 5, b = 4; // The program will only ever add 5 and 4.

Dynamic (Runtime via scanf):

int a, b;
printf("Enter two numbers: ");
scanf("%d %d", &a, &b); // Waits for user to type!

3. The Syntax & Format Specifiers

The term "formatted" means the input data must be read in a specific format dictated by you, the programmer. You do this using a Control String.

scanf("Control String", arg1, arg2);

Instructor's Cheat Sheet for Format Specifiers:

%dInteger
(e.g., 5, -10)
%fFloat
(e.g., 12.53)
%cCharacter
(e.g., 'C')
%lfDouble
(Large decimals)

4. The Memory Map & The Address-of (&) Operator

This is the most crucial concept. When the compiler sees %d, it knows it needs an integer. But where does it put it? That is why we use & (The Address-of Operator).

&a does NOT mean "variable a". It means "the physical memory address of variable a".

int a, b;
scanf("%d %d", &a, &b);

5. Advanced Feature: Assigning Width Limits

You can limit how many digits scanf is allowed to read by putting a number between the % and the letter. The instructor warns this causes abnormal behavior.

scanf("%4d %d", &a, &b);

The leftover 5 is still sitting in the input buffer. The second %d immediately grabs that 5 and puts it into b without waiting for the user!

6. The Secret Return Value

An interesting fact generally unknown to many students: scanf is a function, and like a math function, it evaluates to a specific value.

Fact: scanf returns an integer representing the number of items it successfully read.

int x;
x = scanf("%d %d", &a, &b);
// If the user successfully enters two valid numbers, 
// x will be assigned the value 2.

⚠️ Instructor's Golden Rules

1. NO Commas Inside Quotes: When asking for multiple inputs, write "%d %d". Do NOT write "%d, %d". It causes the compiler to look for literal commas.

2. Always use the ampersand (&): If you write scanf("%d", a); the program will crash. It needs the address (&a), not the variable value.

3. Don't forget the semicolon: Always end your statements with ; or your code won't compile!
