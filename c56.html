The Goal of This Program

In this lesson, the instructor builds a C program to perform three specific tasks on a 2D Array (a Matrix):

  1. Read data from the user into a 2x3 matrix.
  2. Calculate the total sum of all the numbers inside the matrix.
  3. Print the array perfectly formatted as a grid of rows and columns.

⚠️ Instructor Warning: Garbage Values
Notice that sum is explicitly initialized to 0. If you just declare int sum; without giving it a zero, it will hold a random "garbage" value from memory. When you start adding numbers to it, your final math will be completely ruined.

Step 1: Declaration & Reading Input

To read a 2D array, you must use nested loops. The outer loop handles the rows, and the inner loop handles the columns.

int a[2][3], i, j, sum = 0;

// Outer loop (i) runs 2 times for the rows (0 and 1)
for (i = 0; i < 2; i++) {
    // Inner loop (j) runs 3 times for the columns (0, 1, 2)
    for (j = 0; j < 3; j++) {
        // Read input and store it at address &a[i][j]
        scanf("%d", &a[i][j]); 
    }
}

The instructor explains that the inner loop must finish completely (j goes 0, 1, 2) before the outer loop increments (i goes to 1). This ensures Row 0 is completely filled before moving to Row 1.

🖨️

Step 2: Printing & Summing Logic

We use another set of nested loops to print the array and calculate the sum at the exact same time to save processing power.

The Formatting Trick

If we just use printf("%d", a[i][j]), the output will look like a single squished line: 10-13025. We don't want that!

  • \t (Tab Space): We add this after the number so elements in a row are spaced out nicely.
  • \n (Newline): We add this outside the inner loop, but inside the outer loop. This forces the console cursor to drop down to the next line only when a full row is finished.
for (i = 0; i < 2; i++) {
    for (j = 0; j < 3; j++) {
        
        // Print number with a tab space
        printf("%d\t", a[i][j]);
        
        // Calculate running total
        sum = sum + a[i][j];
    }
    // Drop to next line after row finishes
    printf("\n");
}

Step 3: The Dry Run (Tracing the Math)

The instructor carefully updates the variable boxes on the board. Here is exactly how the math trace looks for the first row:

1. Initial State: sum = 0, i = 0

2. Inner loop starts: j = 0
Look at a[0][0]. The value is 10.
Math: sum = 0 + 10. New sum is 10.

3. Inner loop increments: j = 1
Look at a[0][1]. The value is -1.
Math: sum = 10 + (-1). New sum is 9.

4. Inner loop increments: j = 2
Look at a[0][2]. The value is 3.
Math: sum = 9 + 3. New sum is 12.

5. Inner loop finishes. Hits printf("\n"). Outer loop increments i to 1 (Row 1 starts).

This continues for the second row adding 0, 2, and 5 to the total, bringing the final sum to 19.

Watch out! The VS Code Typo

During the practical laptop demonstration, the instructor makes a minor, very common typing error. Make sure your notes reflect the corrected version!

What she typed:
sum = sum = a[i][j]; ❌

This causes a bug because of the double equals sign being used as assignment rather than addition. She spots it while compiling and fixes it immediately.

The correction:
sum = sum + a[i][j]; ✅

The Complete, Final C Program

Here is the exact code from the end of her lesson, ready to be compiled and run.

#include <stdio.h>

int main() {
    // 1. Declare variables
    int a[2][3], i, j, sum = 0;

    printf("Enter elements of Matrix:\n");

    // 2. Read user input
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            scanf("%d", &a[i][j]);
        }
    }

    printf("Matrix is:\n");

    // 3. Print formatting & Math logic
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            // Print with Tab
            printf("%d\t", a[i][j]); 
            
            // Running Total
            sum = sum + a[i][j];     
        }
        // Drop down to next row
        printf("\n"); 
    }

    // 4. Print final sum
    printf("\nSum = %d\n", sum);

    return 0;
}
