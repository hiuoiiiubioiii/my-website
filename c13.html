Operators in C

Complete In-Depth Study Notes & Visualizations

1. What is an Operator?

An operator is simply a symbol that tells the compiler or computer to perform a specific mathematical or logical manipulation on data.

1 + 2 = 3
  • Operator: The + symbol. It tells the computer to perform addition.
  • Operands: The numbers 1 and 2. Operands are the actual data being manipulated.

What is an Expression?

An expression is a sequence of operators and operands that, after processing, results in a single value.

a = 5 + 5

In this expression:

  • 5 and 5 are operands.
  • + and = are operators.
  • a is a variable.

Process: The computer calculates 5 + 5, gets a single value (10), and assigns it to the variable 'a'.

2. Classification of Operators

Operators in C are classified into two main categories:

  • Based on Operands: How many pieces of data the operator requires to work. (Focus of this lesson)
  • Based on Operations: What the operator actually does (Arithmetic, Relational, etc.).

Based on operands, there are three types:

  1. Unary: Requires exactly 1 operand.
  2. Binary: Requires exactly 2 operands.
  3. Ternary: Requires exactly 3 operands.

3. Unary Operators (1 Operand)

These operators only need a single piece of data to function.

1. Unary Minus (-)

Used to change the mathematical sign of a single value. It converts positive to negative, and negative to positive. This is different from the binary minus used for subtraction.

int a = 5;
int b = 10;
int c = a + (-b); // c = 5 + (-10) = -5
a = -b;           // a = -10

2. Increment (++) and Decrement (--)

These operators add 1 or subtract 1 from a variable. Placement matters drastically:

  • Postfix (x++): Written after the variable. The compiler assigns the value first, then increments it.
  • Prefix (++x): Written before the variable. The compiler increments the value first, then assigns it.

3. Logical NOT (!)

Reverses the logical state of an operand (True becomes False, False becomes True).

int x = 11, y = 10;
!(x > y) // (11 > 10) is True. NOT True -> False
!(y > x) // (10 > 11) is False. NOT False -> True

4. Address Of (&)

Retrieves the exact physical memory location (address) of a variable. Used heavily in scanf and with pointers.

5. Sizeof (sizeof())

Tells you how much memory (in bytes) a data type or variable is taking up.

sizeof(int)   // Returns 2 or 4 bytes
sizeof(float) // Returns 4 bytes
int a; 
sizeof(a)     // Returns 2 or 4 bytes

4. Binary Operators (2 Operands)

These require two pieces of data. While detailed in the next lesson, the instructor lists the following categories:

  • Arithmetic: +, -, *, /, % (modulo)
  • Relational: <, >, <=, >=
  • Logical: && (AND), || (OR)
  • Bitwise: &, |, <<, >>, ^, ~
  • Equality: ==, !=
  • Comma: ,
  • Assignment: =

5. Ternary Operator (3 Operands)

Also known as the Conditional Operator. It uses the symbols ? and :. It serves as a rapid shortcut for writing simple if-else statements.

Expression1 ? Expression2 : Expression3;
  • Expression 1: MUST be a condition (evaluates to true or false).
  • If True: Evaluates and returns Expression 2.
  • If False: Skips Expression 2, evaluates and returns Expression 3.

Ternary vs If-Else Equivalent

The instructor demonstrates that the ternary operator does the exact same job as an if-else block, just taking up less space.

// Using Ternary:
X = (a < b) ? a : b;

// Exact Same Logic using If-Else:
if (a < b) {
    X = a;
} else {
    X = b;
}
