To Print Transpose of a Matrix

In-Depth Breakdown of Jenny's Lectures (2D Arrays in C)

1. The Concept: What is a Transpose?

The instructor begins by explaining the fundamental mathematical rule of transposing a matrix:

Rule: To find the transpose of a matrix, you change its Rows into Columns and its Columns into Rows.

If you have a matrix of size 2 x 3 (2 rows, 3 columns), its transpose will be of size 3 x 2 (3 rows, 2 columns).

2. Writing the Logic: The "Index Swap" Trick

We need to read the 2x3 matrix from the user, and then print it as a 3x2 matrix. Here is how she explains the loop logic on the whiteboard.

Step A: Reading the Original Matrix

We use standard nested loops. The outer loop handles the 2 rows, and the inner loop handles the 3 columns.

for(i=0; i<2; i++) {
    for(j=0; j<3; j++) {
        scanf("%d", &a[i][j]);
    }
}

Step B: Printing the Transpose

To print the transpose, we do not need to alter the original array in memory. We just change how we pull data out of it.

  • Change Loop Limits: The new matrix has 3 rows and 2 columns. So, the outer loop runs 3 times, and the inner loop runs 2 times.
  • The Core Trick: Instead of printing a[i][j], we print a[j][i]. This forces the program to read down the column of the original matrix and print it across the row of the console.
// Notice the outer loop is now 3, inner is 2
for(i=0; i<3; i++) {
    for(j=0; j<2; j++) {
        printf("%d\t", a[j][i]);  // Indices swapped here!    }
    printf("\n");
}
