1. Array Theory & Core Rules

Before diving into code, the instructor outlines the fundamental rules of arrays on the whiteboard. Think of an array as a single variable that holds multiple items.

What is an Array?

  • Definition: A collection of more than one data item of the same data type. (You cannot mix int and char).
  • Derived Data Type: It is built upon primitive types (like int, char, float).
  • Memory: Elements are stored in contiguous (continuous) memory locations. There are no gaps between the elements inside the computer's memory.
✅ Advantages of Arrays
  • Fast Access O(1): Accessing any element is incredibly fast using its index (e.g., a[index]).
  • Multidimensional Storage: Easily store data in grids, like 2D matrices or 3D arrays.
⚠️ Drawbacks & Limitations
  • Fixed Size: Once the size is declared, it cannot be changed while the program is running (runtime).
  • Costly Insert/Delete: Inserting an item in the middle requires shifting all subsequent elements, which slows down the computer.
  • No Bound Checking in C: C does not stop you from accessing an index outside the array size. It will just print random memory data, known as a Garbage Value.

Note: Array indexing always starts from 0. An array of size 5 has indices 0, 1, 2, 3, 4.

How does the compiler know the addresses?

If an integer takes 4 bytes of memory, and the Base Address (start) is 100, the next element is exactly 4 bytes away at 104.

🧠 The Secret Math Formula How does the computer instantly find a[3] without searching?

Formula: Base Address + (Index × Size of Data Type)
Example for a[3]: 100 + (3 × 4) = 112. The computer jumps straight to address 112!

3. Reading and Printing Arrays

To interact with all elements inside an array, we use for loops.

Reading Input from the User

for(int i = 0; i < 5; i++) {
    // & symbol gets the specific memory address
    scanf("%d", &a[i]); 
}

As i increases from 0 to 4, scanf places the user's typed number into the corresponding memory slot (as visualized above).

Printing Output (Forward)

for(int i = 0; i < 5; i++) {
    // No & symbol needed here, we just want the value
    printf("%d\n", a[i]);
}

Printing Output (Reverse Order)

To print the array backward (Output: 15, 0, -1, 16, 15), we don't change the array; we just change the loop logic.

// Start at last index (4), stop at first index (0), decrease by 1
for(int i = 4; i >= 0; i--) {
    printf("%d\n", a[i]);
}

4. Full IDE Implementation & Edge Cases

The instructor moves to VS Code to put it all together. Here is the final, clean code snippet demonstrated:

#include <stdio.h>

void main() {
    int a[5], i;
    
    printf("Enter the array elements:\n");
    
    // Loop to Read Data
    for(i = 0; i < 5; i++) {
        scanf("%d", &a[i]);
    }
    
    printf("\nPrinting array elements:\n");

    // Loop to Print Data Forward
    for(i = 0; i < 5; i++) {
        printf("Array element at index %d is: %d\n", i, a[i]); 
    }
}

🚨 Compiler Edge Cases Tested

  • Missing Size: Changing the code to int a[]; immediately throws a compiler error: "error: array size missing". Arrays must have a defined size.
  • Math in Size: Changing the code to int a[3+2]; works perfectly. The compiler evaluates 3+2 before runtime, officially setting the size to 5.
