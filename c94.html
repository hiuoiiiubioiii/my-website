C Programming: Static Variables & For Loops

A complete visual breakdown of function calls inside loop conditions.

🖥️ The Challenge Code

This is the exact code written on the whiteboard. The compiler execution starts from the main() function.

int r()
{
    static int num = 7;
    return num--;
}

int main()
{
    for(r(); r(); r())
        printf("%d\n", r());
    return 0;
}

🧠 Core Concepts (Before we Dry Run)

  • The static keyword: The variable num is initialized to 7 only once. Because it is static, its lifetime lasts throughout the whole program. Every subsequent call to r() ignores the initialization line and holds onto the modified value in memory.
  • Post-decrement (num--): return num--; means the function returns the original value of num first, and then it updates the memory by decrementing it by 1.
  • For Loop Execution Order: for(1. Initialization ; 2. Condition ; 4. Increment) { 3. Body }
  • C Boolean Logic: In C, 0 is evaluated as False. Any other number (positive or negative) is True.

💡 Instructor's Advice

Don't panic when you see weird syntax. Seeing for(r(); r(); r()) might look intimidating because it's not standard like for(i=0; i<10; i++). Just follow the strict execution rules of C step-by-step. Break the problem down, track the variables on paper (like the memory boxes above), and don't skip steps in your head!
