String Conversion in C

In-depth breakdown: Converting Uppercase to Lowercase Letters

Concept 1

Visualizing the Problem

Before writing code, we must define the rules. The objective is to convert only uppercase letters to lowercase. We must completely ignore lowercase letters, numbers, and special symbols.

JENNY ➔ jenny
JeNnY ➔ jenny
JeNnY 123 ➔ jenny 123
#JeNnY + ➔ #jenny +

The Core Rule: We only modify a character if it falls exactly in the range of 'A' to 'Z'.

Concept 2

The Built-In Approach: strlwr()

C has a built-in library function called strlwr() (String Lower) defined inside <string.h>. It takes a pointer to a string, modifies it in memory, and returns the modified string.

// Quick usage of predefined function
char S1[30] = "JayanTi";
strlwr(S1);         
printf("%s", S1); // Output: jayanti
🚨 Important Compiler Warning:

Functions like strlwr(), strupr(), and strrev() are not standard C functions. They are legacy Microsoft C functions. If you compile this on a modern GCC compiler, you will likely get an "undefined reference to strlwr" error. Therefore, we must learn to write our own logic!

Concept 3

The Standard Approach: ASCII Math

Since we can't rely on predefined functions, we have to look at how computers store characters. Every character is stored as an integer known as an ASCII value.

Uppercase

A 65
B 66
... ...
Z 90

Lowercase

a 97
b 98
... ...
z 122

Look closely at the relationship between the tables. There is a mathematical secret here:

97 (a) - 65 (A) = 32

98 (b) - 66 (B) = 32

The Logic: To convert an uppercase letter to a lowercase letter, you simply add 32 to its ASCII value.

Concept 4

Building the Logic (Memory Traverse)

We cannot blindly add 32 to every cell. If we add 32 to 'a', it turns into a special symbol. We must use an if condition with a Logical AND (&&) to ensure the character is an uppercase letter.

for(int i = 0; S1[i] != '\0'; i++) 
{
    // Condition: Is it >= A AND <= Z ?
    if(S1[i] >= 'A' && S1[i] <= 'Z') 
    {
        S1[i] = S1[i] + 32; // Convert to lowercase
    }
}

Instructor Note: You can write S1[i] >= 65 && S1[i] <= 90, but using 'A' and 'Z' inside single quotes is easier because you don't have to memorize the ASCII numbers!

Concept 5

Executing the Dry Run

Let's manually trace the code against our memory array:

  • i = 0 ('J'): Between 'A' and 'Z'? Yes. Add 32. Becomes 'j'.
  • i = 1 ('a'): Between 'A' and 'Z'? No. Condition fails. Skips.
  • i = 2, 3, 4 ('y','a','n'): Condition fails. Skips.
  • i = 5 ('T'): Between 'A' and 'Z'? Yes. Add 32. Becomes 't'.
  • i = 6 ('i'): Condition fails. Skips.
  • i = 7 ('\0'): Loop condition != '\0' becomes false. Loop terminates.

Concept 6

Final Code & Live Execution

Here is the complete, standard C program as written in the live IDE demonstration.

Note: The instructor used gets() to read strings with spaces. While effective for the demo, remember that gets() is unsafe in production code; fgets() is the modern alternative.

#include <stdio.h>
#include <string.h>

void main() {
    char S1[30];
    int i; // Loop variable

    printf("Enter string: ");
    gets(S1); 

    // Traverse string until null character
    for(i = 0; S1[i] != '\0'; i++) 
    {
        // If character is an uppercase letter
        if(S1[i] >= 'A' && S1[i] <= 'Z') 
        {
            S1[i] = S1[i] + 32; // Convert to lowercase
        }
    }

    printf("String is: %s", S1);
}
