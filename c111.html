Initialization & Accessing of Structures

Instructor: Jayanti Khatri Lamba • Subject: C Programming

1. Recap: The Structure Template

Before initializing, we must define the structure template. Think of this as a blueprint.

struct Student
{
    int rollno;
    char name[20];
    float marks;
};
🚨 Critical Rule: You cannot initialize variables inside this definition (e.g., writing int rollno = 1; here is an error). Memory is not allocated until you create an actual variable/object of this structure type.

2. Compile-Time Initialization

This means assigning values to the structure inside your code, before the program runs.

Method A: Full Initialization

You can initialize all members using curly braces {}.

void main() 
{
    // Memory is allocated here
    struct Student S1 = {1, "Jenny", 90.91};
}
🧠 The "Order" Rule: The values inside the curly braces must match the exact order of the data types declared in the structure definition (int, then string, then float).

Method B: Partial Initialization

What happens if you only provide one value?

struct Student S2 = {2};

The compiler assigns 2 to the first member (rollno). Because you left the rest blank, the compiler automatically initializes the remaining members to null or zero.

S2 Memory Block
rollno
2
name
null (\0)
marks
0.000000

Method C: Global Initialization

You can initialize an object right where the structure closes.

struct Student
{
    int rollno;
    char name[20];
    float marks;
} S = {1, "Jenny", 90.91};

3. Accessing Structure Members (The Dot Operator)

You cannot simply write printf("%d", rollno);. The compiler will be confused because it doesn't know whose roll number you want.

The Solution: The Dot Operator (.)
Also known as the Member Access Operator, it links the specific object to its member.

printf("%d", S1.rollno);

4. Copying Structures

Can you copy all data from one student to another quickly? Yes!

S1 = S2;

This takes all the values sitting in the S2 memory block and overwrites the values in the S1 memory block.

S1 (Before)
rollno
1
⟵ COPY ⟵
S2 (Source)
rollno
2
Rule: This only works if S1 and S2 are of the exact same structure type (both are struct Student).

5. Comparing Structures

You cannot compare whole structure objects using relational operators directly.

❌ WRONG:
if (S1 > S2)
if (S1 == S2)

✅ CORRECT WAY: You must compare their individual members using the dot operator.

if (S1.rollno < S2.rollno)
{
    printf("S1 comes first!");
}

6. Run-Time Initialization (Taking User Input)

How to initialize a structure by asking the user to type values while the program is running.

printf("Enter info for S1:");

// Taking integer input (Requires the & symbol!)
scanf("%d", &S1.rollno); 

// Taking string input (Does NOT require & because arrays are pointers)
scanf("%s", S1.name);

// Taking float input (Requires the & symbol!)
scanf("%f", &S1.marks);

You can also combine this into one line:

scanf("%d %s %f", &S1.rollno, S1.name, &S1.marks);

💻 7. IDE Practical Demonstration Code

Here is the full code written in the VS Code IDE at the end of the video. Run this to see initialization and accessing in action!

#include <stdio.h>

struct student {
    int rollno;
    char name[20];
    float marks;
};

void main() {
    // 1. Full Compile-time initialization
    struct student s1 = {1, "jenny", 90};
    
    // 2. Individual Run-time initialization (Taking Input)
    struct student s2;
    printf("Enter info for s2 (Rollno Name Marks):\n");
    scanf("%d %s %f", &s2.rollno, s2.name, &s2.marks);
    
    // 3. Partial Initialization test
    struct student s3 = {3}; // Rollno gets 3, rest become null/0

    // Printing s1
    printf("\nInfo for s1:\n");
    printf("%d %s %f\n", s1.rollno, s1.name, s1.marks);

    // Printing s2
    printf("\nInfo for s2:\n");
    printf("%d %s %f\n", s2.rollno, s2.name, s2.marks);
    
    // Printing s3 (Notice name is blank, marks are 0.000000)
    printf("\nInfo for s3:\n");
    printf("%d %s %f\n", s3.rollno, s3.name, s3.marks);
}
