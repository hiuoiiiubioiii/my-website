Pre-defined & Conditional Macros

1. Quick Recap: What is a Macro? Fundamentals

Before jumping into pre-defined macros, let's establish exactly what a macro is doing under the hood.

  • Definition: A macro is a segment of code that is replaced by its assigned value.
  • When does this happen? During the pre-processing phase. This happens before the code actually compiles.
Two previous commands we've learned:

👉 #define : Used to define a macro (constants, variables, or functions).
👉 #undef : Used to un-define a macro so it can no longer be used.

2. The 5 Core Pre-Defined Macros Syntax

C provides built-in macros that you can use directly without defining them yourself. They are extremely useful for debugging and generating logs.

⚠️ Crucial Syntax Note: All pre-defined macros are written in ALL CAPS and must start and end with TWO UNDERSCORES (__).

Here is how the instructor wrote them on the board, including their proper format specifiers (%s for strings, %d for integers):

1. Current Date
printf("Date: %s", __DATE__);
↳ Prints as String literal format: MMM DD YYYY (e.g., Apr 25 2022)

2. Current Time
printf("Time: %s", __TIME__);
↳ Prints as String literal format: HH:MM:SS

3. Current File Name
printf("File: %s", __FILE__);
↳ Prints the name of the file currently executing (e.g., predefinedmacro.c)

4. Current Line Number
printf("Line: %d", __LINE__);
↳ Prints the exact line number in the source code as an Integer.

5. ANSI Standard Check
printf("ANSI: %d", __STDC__);
↳ Returns Integer 1 (True) if the C compiler complies with ANSI standards.

3. Conditional Macros (Directives) Logic

You already know if-else statements, but Conditional Macros are different.

  • Standard if-else runs during program execution (runtime).
  • Conditional Macros run before compilation (pre-processing). They tell the compiler whether to include or completely skip entire blocks of code.

Available Directives: #ifdef, #ifndef, #if, #else, #elif, #endif

Example A: #ifdef (If Defined)
#ifdef MACRO
    // Code goes here
#endif
↳ Preprocessor asks: "Has MACRO been defined earlier?" If True, include this code. If False, the compiler never even sees this code.

Example B: #ifndef (If NOT Defined)
#ifndef MACRO
    // Code goes here
#endif
↳ The exact opposite. If MACRO has NOT been defined, include this code.

Example C: #if / #else
#if expression
    // Code block 1
#else
    // Code block 2
#endif

Here is the exact code the instructor ran in Visual Studio Code to demonstrate the output of the pre-defined macros.

#include <stdio.h>

int main() {
    
    printf("Date : %s \n", __DATE__);
    printf("Time : %s \n", __TIME__);
    printf("File : %s \n", __FILE__);
    printf("Line : %d \n", __LINE__); // Currently on line 8
    printf("ANSI : %d \n", __STDC__);
    
    return 0;
}
