Sum of Individual Rows & Columns in a Matrix

In this lesson, we will build a C program that calculates and prints the sum of each row and each column independently in a 3x3 matrix. Scroll down to master the logic step-by-step.

1. The Two "Tricks" (The Logic)

To solve this, we define two loop counters: i (outer loop) and j (inner loop). We also need variables to store our running totals: sumRow and sumCol.

Summing Rows

We use standard traversal: a[i][j].

Here, i (Row 0) stays the same while j increments (Col 0, Col 1, Col 2), dragging us horizontally.

Summing Columns

We swap the indices: a[j][i].

Here, the column index i stays the same, while the row index j increments, pulling us vertically down the matrix.

CRITICAL INSTRUCTOR WARNING: You MUST reset sumRow = 0; and sumCol = 0; inside the outer loop, right before the inner loop starts. If you don't, Row 1's sum will be added to Row 0's sum instead of starting fresh!

2. The Whiteboard "Dry Run"

Let's manually trace the code for the first row and column to see how the computer updates its memory.

Outer Loop Starts: i = 0

We initialize our trackers inside the outer loop.

i = 0 | sumRow = 0 | sumCol = 0

Inner Loop: j = 0

Summing the very first element (Row 0, Col 0 is '1').

sumRow = 0 + a[0][0] ➔ 1
sumCol = 0 + a[0][0] ➔ 1

Inner Loop: j = 1

Row moves right (a[0][1] = 2). Col moves down (a[1][0] = 2).

sumRow = 1 + a[0][1] ➔ 3
sumCol = 1 + a[1][0] ➔ 3

Inner Loop: j = 2

Row moves right (a[0][2] = 1). Col moves down (a[2][0] = 0).

sumRow = 3 + a[0][2] ➔ 4
sumCol = 3 + a[2][0] ➔ 3

Inner Loop Finishes

The code prints the result and moves to the next outer loop iteration i = 1. The sums are immediately reset to 0!

OUTPUT: sumRow=4, sumCol=3

3. Final Practical Implementation

Here is the final C Code written in VS Code by the instructor. Notice the formatting adjustments to print the matrix nicely before showing the sums.

#include<stdio.h>

int main() {
    int a[3][3], i, j, sumRow, sumCol;
    
    // 1. Take input from user
    printf("Enter matrix:\n");
    for(i=0; i<3; i++) {
        for(j=0; j<3; j++) {
            scanf("%d", &a[i][j]);
        }
    }
    
    // 2. Print the matrix visually
    printf("Matrix is:\n");
    for(i=0; i<3; i++) {
        for(j=0; j<3; j++) {
            printf("%d\t", a[i][j]); // \t adds a tab space
        }
        printf("\n"); // next line after a row
    }
    
    // 3. Calculate and print sums
    for(i=0; i<3; i++) {
        // Critical Reset Step
        sumRow = 0;
        sumCol = 0;
        
        for(j=0; j<3; j++) {
            sumRow = sumRow + a[i][j]; // Normal traversal
            sumCol = sumCol + a[j][i]; // Swapped traversal
        }
        
        printf("\nsumRow=%d, sumCol=%d", sumRow, sumCol);
    }
    
    return 0;
}
