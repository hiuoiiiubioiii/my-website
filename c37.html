The `while` Loop in C

Comprehensive Study Notes & Visualizations based on Jenny's Lectures

📌 1. Introduction to the `while` Loop

The while loop is used to repeat a block of code as long as a specified condition remains true. It is almost identical in function to a for loop, but the way we write it (the syntax) is different.

  • Key Concept: It is an Entry-Controlled Loop.
  • What this means: The condition is checked first, before entering the loop body. If the condition is false from the very beginning, the code inside the loop will never execute. Control exits immediately.

✍️ 2. General Syntax & Rules

In a for loop, initialization, condition, and updating all happen on one line. In a while loop, they are separated into three different places.

Initialization;              // 1. Setup before the loop starts

while (Condition)            // 2. The Termination Condition
{
    // Body of the loop
    Statement 1;             // 3. What the loop actually does
    Statement 2;
    
    Update/Modify;           // 4. Increment/Decrement (e.g., i++)
}

Important Rules:

  • Condition is Mandatory: You must put a termination condition inside the (). Without it, or without a proper update statement, you will create an infinite loop.
  • Curly Braces {}: If you have only one statement in the loop body, curly braces are optional. If you have multiple statements, they are required.

For Loop

for(i=1; i<=10; i++)

Everything is clustered in one neat package at the top.

While Loop

i = 1;
while(i<=10)
{ ... i++; }

Spread out. Good for when you don't know the exact number of iterations.

💻 4. Practical Example: Printing 1 to 10

We don't need user input for this. We know exactly where to start (1), where to end (10), and the step (+1). Because i is the standard "loop girlfriend", we will use i as our variable.

#include <stdio.h>
#include <conio.h>

void main()
{
    int i = 1;              // Initialization
    
    while (i <= 10)         // Condition Check
    {
        printf("%d \n", i); // Statement (Print current value, \n for new line)
        i++;                // Update (Increase by 1)
    }
    
    getch();                // Wait for key press to close
}

Step-by-Step Breakdown:

  • Start: Compiler reads int i = 1;. It creates a box in memory and puts 1 inside.
  • Check 1: while (1 <= 10). TRUE. Prints 1. i++ changes memory from 1 to 2.
  • Check 2: while (2 <= 10). TRUE. Prints 2. i++ changes memory to 3.
  • (Fast forward...)
  • Check 10: while (10 <= 10). TRUE. Prints 10. i++ changes memory to 11.
  • Check 11: while (11 <= 10). FALSE! The loop instantly terminates. Control goes to getch().

🎓 6. Logic Building Tips for Students

How do you approach a looping problem when you are a beginner? The instructor gave excellent advice for building logic:

  • Analyze the desired output first: Look at what needs to be printed (e.g., 1 to 10). From this, identify your 3 pillars:
    • Where do I start? (Initialization)
    • Where do I end? (Condition)
    • What is the step? (Increment/Decrement)
  • Why use loops? If you want to print 1 to 10, you *could* write printf ten times. But for repetition, loops save time and lines of code.
  • The Golden Rule: Do not type directly into the computer immediately! Get a pen and paper. Write the logic down manually. "Dry run" it with your own hands. This is vital for university exams and placement interviews, where you will be asked to write logic on a whiteboard or paper without a computer!
  • Start Simple: Try simple programs first before jumping to complex loop tricks and nested loops.
