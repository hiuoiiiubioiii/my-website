1. The Core Concept: How Strings are Stored

Before writing code, we must understand how C handles strings in memory. A string is essentially an array of characters.

  • The Null Character: Every valid string in C is terminated by a special character: '\0' (Null). This tells the compiler where the string ends.
  • Length Definition: The length of a string is the number of characters before the null character. The null character itself is not counted.

2. Method 1: Using the Predefined Function strlen()

C provides a built-in library function to do this work for us. Because it is "predefined", its logic is already written in a header file.

  • Header Required: #include <string.h>
  • Function Signature: unsigned int strlen(const char *str);
Instructor Insight: Why 'unsigned int'?
The strlen() function returns an unsigned integer because a length can never be a negative number. It will always be 0 or greater. Also, passing the array name into the function passes a pointer to the base address (e.g., memory address 1000).
// Method 1: Using predefined function #include <stdio.h> #include <string.h> // Required for strlen() int main() { char name[30]; int count = 0; printf("Enter name: "); gets(name); // 'gets' allows reading strings with spaces // strlen counts characters until it hits '\0' count = strlen(name); printf("String length is %d", count); return 0; }

3. Method 2: Custom Logic (Under the Hood)

What if we don't want to use the <string.h> library? We can write our own logic to find the length. This is exactly what strlen() is doing behind the scenes.

The Logic: We initialize a counter (count = 0) and an index (i = 0). We use a while loop to check each character in the array. If the character is NOT '\0', we increase the count and move to the next index.

// Method 2: Custom Logic (No string.h) #include <stdio.h> int main() { char name[30]; int count = 0; int i = 0; printf("Enter name: "); gets(name); // Custom Length Logic while(name[i] != '\0') { count++; // Increase length count i++; // Move to the next index } printf("String length is %d", count); return 0; }

What about Spaces? (e.g., "Jayanti Khatri")
If the user inputs a space, does the loop break? No.
The ASCII value of a space character ' ' is 32. The ASCII value of the null character '\0' is 0. Because 32 is not equal to 0, the condition remains True. The loop will count the space as a valid character, resulting in a length of 14.
