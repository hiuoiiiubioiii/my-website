🔍 1. Introduction to String Comparison

In C, we cannot compare strings using the standard == operator because strings are arrays (pointers). Instead, we compare them using Lexicographical Comparison (character-by-character from left to right).

How it works:

  • Example 1: computer vs computer. Result: Equal.
  • Example 2: Computer vs computer. Result: Not Equal. The very first characters ('C' vs 'c') are different due to case sensitivity.
Instructor's Note: There are two ways to do this. We can use the predefined C function strcmp(), or we can write our own custom logic using loops. We will learn both!

⚙️ 2. Deep Dive: The strcmp() Function

To use the built-in function, you must include the header file <string.h>.

The Prototype (Under the hood)

int strcmp(const char *str1, const char *str2);
  • char *str1: A pointer to the base address of the first string.
  • const: Means the content at the memory location cannot be changed by this function. It makes the string read-only during comparison.
  • int: The function returns an integer based on the comparison.

Understanding the Return Values

The function subtracts the ASCII value of the characters. It returns three possible values:

Return Value Meaning Condition
0 (Zero) Strings are identical Both strings match perfectly, including case.
+ve (Positive) str1 > str2 The first unmatched character in str1 has a larger ASCII value than in str2.
-ve (Negative) str1 < str2 The first unmatched character in str1 has a smaller ASCII value than in str2.

ASCII Logic Example

If we compare str1 = "computer" (lowercase 'c') with str2 = "Computer" (uppercase 'C'):

ASCII of 'c' = 99
ASCII of 'C' = 67

Math: 99 > 67. Therefore, `str1` is greater.
Result: Returns a Positive (+ve) value.

💻 3. Basic Program Using strcmp()

Here is how we use the predefined function in a real C program.

#include <stdio.h>
#include <string.h>

void main() {
    char s1[] = "Hello";
    char s2[] = "Welcome";
    int value; // To store the return result

    value = strcmp(s1, s2);

    if (value == 0) {
        printf("Same");
    } else {
        printf("Not same");
    }
}

The Logic Breakdown

We need to loop through the indices. But when does the loop stop? We stop when either string reaches its Null character \0.

Crucial Condition: We use the Logical OR operator (||). The loop continues as long as:
s1[i] != '\0' || s2[i] != '\0'

Dry Run (Tracing the code)

  • i = 0: Compare s1[0] ('H') and s2[0] ('H'). They match. Loop continues.
  • i = 1: Compare s1[1] ('e') and s2[1] ('i'). They DO NOT match (Highlighted in red above).
  • At index 1, we set a flag = 1 and instantly break; out of the loop. There is no need to check the rest!

👨‍💻 5. Final Custom Implementation (IDE)

Here is the full, complete code without using predefined functions, written in the IDE.

#include <stdio.h>

void main() {
    int flag = 0, i; // 0 means we assume they are the same
    char s1[30], s2[30];

    printf("Enter string 1: ");
    gets(s1);
    printf("Enter string 2: ");
    gets(s2);

    // Loop until BOTH strings reach null
    for(i = 0; s1[i] != '\0' || s2[i] != '\0'; i++) {
        
        // If a mismatch is found
        if(s1[i] != s2[i]) {
            flag = 1; // Flag 1 means "Not Same"
            break;    // Stop looping
        }
    }

    // Final Check
    if(flag == 0) {
        printf("same");
    } else {
        printf("not same");
    }
}
