Pointers to Structures in C

Master Study Guide & Visual Notes based on Jenny's Lectures

1 Defining the Structure (Review)

Before creating a pointer, we must define the structure. A structure is a user-defined data type that acts as a blueprint.

struct student {
    int rollno;
    char name[20];
    float marks;
};
Instructor's Audio Note: struct student is the complete name of this new data type. rollno, name, and marks are called structure members. They are not variables yet; no memory is allocated just by defining this blueprint.

2 Declaring Structure Variables

To use the structure, we must declare a variable of its type. Where you place this declaration dictates its scope.

Method A: Global Variable

struct student s;  // Declared outside of main()

void main() {
    // Code here
}

Meaning: s is a global variable. It can be accessed and used by main() and any other function in your program.

Method B: Local Variable

void main() {
    struct student s; // Declared inside main()
}

Meaning: s is local to main(). It can only be used within the main() function.

3 The Golden Rule of Pointers

A pointer is a special variable that stores the memory address of another variable.

The Rule: The data type of the pointer must exactly match the data type of the variable whose address it is storing.

✅ VALID: Both are 'int'
int a;
int *ptr = &a;
❌ ILLEGAL: Data types do not match.
float b;
int *ptr = &b;

4 Declaring a Pointer to a Structure

Applying the Golden Rule: If we want a pointer to store the address of our structure variable s, the pointer's data type must be exactly struct student.

void main() {
    // 1. Variable of type 'struct student'
    struct student s;           
    
    // 2. Pointer of type 'struct student'
    struct student *ptr;        
    
    // 3. Storing the address of 's' into 'ptr'
    ptr = &s;                   
}

5 Visualizing Memory Allocation

This is what the computer does behind the scenes when we write:
struct student s = {1, "Jenny"}; ptr = &s;

Pointer (ptr)
1000 Value stored Address: 2000
Structure Variable (s)
1 rollno 1000
"Jenny" name 1004
0.0000 marks 1024
  • Block 1 (s): Memory is allocated contiguously. If the base address is 1000, rollno is at 1000, name starts at 1004, and marks starts at 1024.
  • Block 2 (ptr): ptr is a separate variable (e.g., at address 2000). Because we wrote ptr = &s;, it stores the base address 1000.

6 Accessing Structure Members

How do we print these values? We have three methods:

Method 1: Using the Variable Name (Dot Operator)

We use the dot operator (.) when accessing members through the standard variable name.

printf("Rollno is %d", s.rollno);

Method 2: Using the Pointer (Arrow Operator)

Since ptr holds the address, we cannot write ptr.rollno. We use the Arrow Operator (->) (a minus sign followed by a greater-than sign).

printf("Rollno is %d", ptr->rollno);

Method 3: Dereferencing + Dot Operator

This is an alternative to the arrow operator.

printf("Rollno is %d", (*ptr).rollno);
Why does Method 3 work?
1. ptr = 1000
2. *ptr means "Value at address 1000". The value at address 1000 is the actual structure s.
3. Therefore, (*ptr) is exactly equivalent to s.
4. So, (*ptr).rollno is exactly the same as typing s.rollno.

Warning: The parentheses () are mandatory due to C's operator precedence rules!

7 Full IDE Implementation

This is the complete, runnable C code written by the instructor in Visual Studio Code.

#include <stdio.h>

// 1. Define the Structure
struct student {
    int rollno;
    char name[20];
    float marks;
};

void main() {
    // 2. Declare and initialize the structure variable
    struct student s1 = {1, "Jenny", 90.0};
    
    // 3. Declare the structure pointer
    struct student *ptr;
    
    // 4. Assign the address of s1 to ptr
    ptr = &s1;

    // 5. Printing using the Pointer Arrow Operator
    printf("Info for s1:\n");
    printf("%d %s %f", ptr->rollno, ptr->name, ptr->marks);

    /* 
    ALTERNATIVE WAY TO PRINT:
    printf("%d %s %f", (*ptr).rollno, (*ptr).name, (*ptr).marks);
    */
}
Terminal Output:
Info for s1:
1 Jenny 90.000000

At the very end of the video, the instructor gives a challenge:

Instead of "compile-time initialization" (hardcoding {1, "Jenny", 90.0}), write a program that uses printf to ask the user to enter the data, and use scanf along with the pointer to take the inputs at run-time.

Cheat Sheet / Hint:
To use scanf with a pointer arrow operator, simply add the ampersand (&) before it (except for strings, which don't need the ampersand).

Example: scanf("%d", &ptr->rollno);
