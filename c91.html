Functions WITH Arguments & WITH Return Type

In-depth lecture breakdown & whiteboard visualizations

1. Concept Overview

This is the 4th and final classification of functions in C. In this structure, data flows in both directions:

  • With Arguments: The calling function (e.g., main()) passes actual data (values) to the user-defined function.
  • With Return Type: The user-defined function performs a calculation and sends a final result back to the calling function.
Instructor's Rule: Because data is being passed into the function from main, you do not need printf or scanf inside the user-defined function. The input is already provided!

2. Step-by-Step Code Construction

Step A: Function Declaration (Prototype)

Tell the compiler what the function looks like before `main()`. We want the sum of two integers, so the return type is int, and it accepts two int arguments.

int sum(int, int);

Step B: The Main Function (Calling)

We ask the user for two numbers, store them in `x` and `y`, and pass them to our function. We must also create a variable `c` to "catch" the answer returning from the function.

void main() {
    int x, y, c;
    
    printf("Enter x and y: ");
    scanf("%d %d", &x, &y);
    
    // Call the function, pass x and y, and catch the result in c
    c = sum(x, y); 
    
    printf("Sum = %d", c);
}

Step C: The Function Definition (The Logic)

Here, we must give the arguments variable names (a and b) to hold the incoming data. We process the sum and use the return keyword.

int sum(int a, int b) {
    int s = 0;   // Local variable for processing
    s = a + b;
    return s;    // Send the answer back to main

    // Alternate Syntax mentioned by instructor:
    // return a + b;
}

3. Visualizing Memory (The "Dry Run")

How does the computer actually execute this? The instructor drew memory blocks on the whiteboard. Let's assume the user enters 5 and 7.

main() Memory Block
x 5
y 7
c ⏳

c = sum(x, y); pauses main() and passes copies of 5 and 7 down.

⬇️ Call by Value ⬇️
sum() Memory Block
a 5
b 7
s 12

s = a + b; computes 12. return s; sends 12 back up!

⬆️ Return Value (12) ⬆️
Crucial Scope Rule: The variables x, y, c belong ONLY to main(). The variables a, b, s belong ONLY to sum(). You cannot write printf("%d", s) inside main, because s is destroyed the moment the sum function finishes running!

4. Complete Execution & Homework

Putting it all together, here is the final code that the instructor ran inside VS Code during the demonstration:

#include <stdio.h>

int sum(int, int);

void main() {
    int x, y, c;
    printf("Enter x and y: ");
    scanf("%d %d", &x, &y);
    
    c = sum(x, y);
    printf("Sum = %d", c);
}

int sum(int a, int b) {
    return a + b;
}
