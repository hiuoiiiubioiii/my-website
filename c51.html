Counting Even & Odd Numbers in C

A step-by-step breakdown of array iteration, memory visualization, and loop optimization.

🛠️ 1. Setup & Variable Declarations

To start, we need to set up our memory container (the array) and variables to keep count.

int a[10];
int i;
int even = 0, odd = 0;
⚠️ The "Garbage Value" Trap!
You MUST initialize your counters to 0 (even = 0, odd = 0). If you don't, C assigns them random "garbage values" (like 24198). If you add 1 to a garbage value, your final count will be completely ruined!

🧠 3. The Core Logic (The Modulo Operator)

To check if a number is even or odd, we use the Modulo Operator (%). It divides a number and gives us the remainder.

The Golden Rule:
If Number % 2 == 0 ➡️ The number is EVEN.
If Number % 2 != 0 ➡️ The number is ODD.

Tracing the first few loops:

i = 0 ➡️ a[0] is 1 ➡️ 1 % 2 = 1 Condition FALSE. odd++
i = 1 ➡️ a[1] is 2 ➡️ 2 % 2 = 0 Condition TRUE. even++
i = 2 ➡️ a[2] is 4 ➡️ 4 % 2 = 0 Condition TRUE. even++

By the end of this loop, the computer will have tallied up even = 6 and odd = 4 based on our array above.

⚡ 4. Code Optimization: 2 Loops vs 1 Loop

Initially, the instructor writes a loop to scanf the inputs, and then a second separate loop to check for even/odd.

The "Aha!" Moment: Why read all 10 numbers first, and then loop through them all over again to check them? We can do this in ONE loop! We can check if the number is even or odd the very second the user presses 'Enter'.

// OPTIMIZED: Doing everything in one single loop!
for(i = 0; i < 10; i++) {
    
    // 1. Grab the input from the user
    scanf("%d", &a[i]);

    // 2. IMMEDIATELY check if it's even or odd
    if(a[i] % 2 == 0) {
        even++;  // Same as even = even + 1
    } else {
        odd++;   // Same as odd = odd + 1
    }
}

💻 5. The Final, Complete VS Code Implementation

Here is the final code the instructor writes in VS Code, combining all the concepts we learned above, alongside the print statements to display the results.

#include <stdio.h>

void main() {
    int a[10], i;
    int even = 0, odd = 0;

    printf("Enter array elements:\n");

    for(i = 0; i < 10; i++) {
        scanf("%d", &a[i]);
        
        if(a[i] % 2 == 0) {
            even++;
        } else {
            odd++;
        }
    }

    // Print results OUTSIDE the loop
    printf("\nTotal even numbers are: %d", even);
    printf("\nTotal odd numbers are: %d", odd);
}
