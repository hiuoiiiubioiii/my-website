1. The Problem Code

Here is the exact C code snippet provided on the whiteboard. Copy this into your physical notes.

int incr(int i)
{
    static int count = 0;
    count = count + i;
    return (count);
}

main()
{
    int i, j;
    for(i = 0; i <= 4; i++)
        j = incr(i);
}

3. Crucial Concept: The static Keyword

Before tracing, the instructor stops to explain the most important line: static int count = 0;

  • Lifetime: A static variable's memory lasts for the entire duration of the program, not just when the function is running.
  • Initialization: It is initialized ONLY ONCE. If you call the function a second time, the computer completely ignores the initialization line.
  • Memory: It remembers its previous value between function calls. Normal local variables (like i) are destroyed and recreated.
Iteration 1: i = 0

Executing the First Loop

Execution begins in main(). i is set to 0. Is 0 <= 4? Yes. We call incr(0).

Inside incr(), local i receives 0. The static variable count is created and initialized to 0. We do the math: count = 0 + 0 = 0. The function returns 0 back to main(), storing it in j.

Inside incr(), local i is recreated and receives 1. CRITICAL: The line static int count = 0; is SKIPPED. count remembers its value is 0.

Math: count = 0 + 1 = 1. Returns 1 to j.
