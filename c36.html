Print the Table of a Number

A complete, step-by-step logic and memory breakdown of loops in C. Scroll down to begin. 👇

1 The "Think Before You Code" Phase

Before writing a single line of C code, the instructor emphasizes understanding the desired output to figure out the logic.

// Expected Output (if user enters 2)
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
...
2 x 10 = 20
🧠

The Constant (number): The number '2' at the beginning stays the same. This will be the variable containing user input.

🔄

The Counter (i): The multiplier (1, 2, 3... 10) increases by exactly 1 each time. Because this is repetitive, we must use a loop.

🧮

The Result (a): The final answer (2, 4, 6...) is the calculation of number * i. We need a variable to store this result in memory.

2 Writing the Code

Here is exactly what she writes on the whiteboard, translated into clean C syntax. Read the inline annotations to understand what each block does.

#include <stdio.h> #include <conio.h> void main() { // 1. Declare 3 memory blocks int number, i, a; // 2. Ask user for input and store it at address (&) of 'number' printf("Enter a number:"); scanf("%d", &number); // 3. The Core Logic: Start at 1, go up to 10, add 1 each time for (i = 1; i <= 10; i++) { // Math: Multiply user input by the loop counter a = number * i; // Print the result. '\n' is crucial to print a vertical list! printf("%d\n", a); } getch(); }

4 The Dry Run (Execution)

Loop Cycle 1

  • i initializes to 1.
  • Condition Check: Is 1 <= 10? TRUE
  • Math: a = 2 * 1 (a becomes 2).
i
1
a
2
> Output: 2

Loop Cycle 2

  • i++ triggers. i becomes 2.
  • Condition Check: Is 2 <= 10? TRUE
  • Math: a = 2 * 2 (a becomes 4).
i
2
a
4
> Output: 4
... loops 3 through 9 execute ...

Loop Cycle 10

  • i++ triggers. i becomes 10.
  • Condition Check: Is 10 <= 10? TRUE
  • Math: a = 2 * 10 (a becomes 20).
i
10
a
20
> Output: 20

Loop Termination

  • i++ triggers. i becomes 11.
  • Condition Check: Is 11 <= 10? FALSE
  • The condition fails. The program jumps out of the loop, hits getch(), and the program terminates.
