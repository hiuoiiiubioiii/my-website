Nested Do-While Loop in C

Instructor: Jayanti Khatri Lamba | Complete Visual Breakdown

1. What is Nesting?

Nesting means placing one loop inside the body of another loop. "Nested do-while" specifically means writing a do-while loop inside another do-while loop.


Instructor's Tip: You are not restricted to nesting the same type of loop. You can mix and match a for loop, a while loop, and a do-while loop as deep as your program needs. 
for( ... ) {
    while( ... ) {
        do {
            // This is perfectly valid!
        } while( ... );
    }
}

2. The Syntax Anatomy

Here is how the instructor outlines the structure of a nested do-while loop on the whiteboard:

do {
    // 1. Outer loop statements (Executes before inner loop)
    
    do {
        // 2. Inner loop statement block
        // (Executes completely for every single outer iteration)

    } while( Inner_Condition );

    // 3. Outer loop statements (Executes after inner loop finishes)

} while( Outer_Condition );

Crucial Rule: Because it is an exit-controlled loop, the condition is checked at the bottom. This guarantees the inner loop statements will execute at least once, even if the condition is false on the very first try.

3. Visual Logic Flow

How the computer reads the nested loop top-to-bottom:

Outer Loop Starts
↓
Execute Outer Statements
↓
Inner Loop Starts
↓
Execute Inner Statements
↓
Is Inner Condition True?

(If YES, loop back to Inner Statements)

↓ (If NO)
Execute Remaining Outer Statements
↓
Is Outer Condition True?

(If YES, loop all the way back to the top)

↓ (If NO)
EXIT Nested Loop

4. Dry Run / Program Trace

The instructor writes a C program on the board and traces the memory state step-by-step. Let's trace it exactly as she did.

int i = 1, j = 3;

do {
    do {
        printf("%d ", j);
        j--;
    } while (j > 0);

    i++;
    printf("%d ", i);

} while (i < 4);

Execution Trace:

▶ Outer Loop: Iteration 1

Variable i
1
Variable j
3 2 1 0
  • Enter Inner Loop: Prints 3, decreases j to 2. (2>0 is True).
  • Prints 2, decreases j to 1. (1>0 is True).
  • Prints 1, decreases j to 0. (0>0 is False -> Exit inner loop).
  • Back to Outer: i++ makes i = 2. Prints 2.
  • Check outer condition: i < 4 (2 < 4 is True).
Output: 3 2 1 2

▶ Outer Loop: Iteration 2

Notice `j` does NOT reset to 3. It starts this iteration at 0!
Variable i
1 2 3
Variable j
0 -1
  • Enter Inner Loop: Because it's a do-while, it enters *even though* j is not > 0.
  • Prints 0, decreases j to -1. (-1>0 is False -> Exit inner loop).
  • Back to Outer: i++ makes i = 3. Prints 3.
  • Check outer condition: i < 4 (3 < 4 is True).
Output: 3 2 1 2 0 3

▶ Outer Loop: Iteration 3

Variable i
1 2 3 4
Variable j
-1 -2
  • Enter Inner Loop: Enters automatically again.
  • Prints -1, decreases j to -2. (-2>0 is False -> Exit inner loop).
  • Back to Outer: i++ makes i = 4. Prints 4.
  • Check outer condition: i < 4 (4 < 4 is False -> Exit entire program).
Final Output: 3 2 1 2 0 3 -1 4

5. Homework Assignment

At the end of the lecture, the instructor updates the code using pre/post increment operators inside the printf statements. She asks you to dry-run this on paper and comment the output.

int i = 1, j = 3;

do {
    do {
        // Change 1: Pre-decrement
        printf("%d ", --j);
    } while (j > 0);

    // Change 2: Post-increment
    printf("%d ", i++);

} while (i < 4);
Hint for Homework:
--j: Decreases the value before printing it.
i++: Prints the current value, and increases it after printing!
