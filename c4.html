1. High-level lang

Abstraction from Hardware

C is a high-level language. This means you, the programmer, focus on the logic and the program itself, rather than the low-level machine details.

  • You don't need to worry about the CPU architecture or specific hardware.
  • It uses English-like words and mathematical notation.
  • It handles variables, constants, and data types in a way that is human-readable.
Board Visual
2. small lang having 32 keywords

Simplicity & Size

C is considered a "small" language. It has a very limited vocabulary.

Fact: C only has 32 keywords. Compared to languages like Java or C++, this makes C syntactically easier to learn and memorize.
Board Visual
3. Core lang. as many other Prog. lang. are dependent on C

The Mother Language

C is a core, foundational language. Many modern languages (C++, Java, Python) derive their syntax and concepts directly from C.

"If you learn C, learning other languages becomes much easier."

Board Visual
4. Portable

Machine Independence

Programs written in C are portable. This means you can write a program on one computer and run the source code on a completely different machine.

While you might need minor modifications occasionally, generally, the logic remains the same across platforms.

Board Visual
5. built-in functions & operators

Standard Libraries

C comes with a rich library of pre-defined functions. You don't have to reinvent the wheel.

  • If you need to print to the screen? Use printf().
  • Need math? Use the math library.

You access these by including header files at the top of your program.

Board Visual
6. Structured lang Modular

Organization via Functions

C allows you to break a large program into smaller blocks called Functions.

Example: If you are calculating the area of different shapes, you don't write one giant block of code. You create:

  • Module 1: Area of Triangle
  • Module 2: Area of Rectangle
  • Module 3: Area of Circle

This is called Modular Programming. It makes code easier to debug and read.

Board Visual
7. Pointers

Direct Memory Access

This is one of C's most powerful features. Pointers allow you to directly interact with computer memory.

A pointer is a variable that stores the address of another variable. This allows for efficient memory management and system-level programming.

Board Visual
8. extensible lang

Adding Your Own Features

C is extensible. You are not limited to the standard library provided by C.

You can create your own functions and add them to the C library. Essentially, you can "extend" the language to suit your specific needs.

Board Visual
9. compilation & execution is faster

Performance

Because C is a Compiled Language (not interpreted) and it is a "small" language with less overhead, it is extremely fast.

The time required to compile and execute a C program is significantly less than languages like Python or Java.

Board Visual
10. Dynamic Memory allocation
free( )

Runtime Memory Management

C allows you to allocate memory while the program is running (using malloc and calloc).

Important: You can also free up memory at any time using the free() function. This makes C very efficient with system resources.
Board Visual
11. are Sensitive
a   A

Case Sensitivity

C distinguishes between uppercase and lowercase letters.

The variable 'a' is treated as completely different from the variable 'A'. You must be careful with capitalization.

Board Visual
12. Embedded System

System Programming

C is used to develop Embedded Systems. These are the computer systems inside devices like microwaves, refrigerators, and washing machines.

Because it is close to the hardware, C is also used to write Operating Systems and drivers.

Important Clarification
.exe (Windows)
.dmg (Mac)

Portability vs. Platform Dependency

The instructor makes a crucial distinction here:

  1. Source Code (Portable): The text code you write is portable. It can move between Windows and Mac.
  2. Object Code (Dependent): When you compile code on Windows, it creates an .exe file. This file cannot run on Mac or Linux.

Conclusion: The language is portable, but the compiled executable file is platform-dependent.
