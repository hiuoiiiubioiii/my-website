State 2: The Logic Execution

During the third loop, when i = 0:

  1. The computer looks at sumArr[0] = arr1[0] + arr2[0].
  2. It fetches values from memory addresses 100 and 200.
  3. Calculates 10 + 0 = 10.
  4. It overwrites the garbage value at memory address 1000 with the number 10.

💻 4. The Final Code (Live IDE Version)

At the end of the video, the instructor switches to VS Code. She changes the array names to a, b, and c for simplicity, and adds print statements to guide the user.

💡 Instructor's Syntax Tip If there is only one statement inside a for loop, you do not need to use curly braces {}. However, in our third loop, we calculate and print, so curly braces are mandatory there! 
#include <stdio.h>

int main() {
    // 1. Declare three arrays of size 5, and a loop counter 'i'
    int a[5], b[5], c[5];
    int i;

    // 2. Input for First Array
    printf("Enter first array elements:\n");
    for(i = 0; i < 5; i++) {
        scanf("%d", &a[i]);
    }

    // 3. Input for Second Array
    printf("Enter second array elements:\n");
    for(i = 0; i < 5; i++) {
        scanf("%d", &b[i]);
    }

    // 4. Calculating Sum and Printing (Requires curly braces for 2 statements)
    for(i = 0; i < 5; i++) {
        // Core logic: overwrite garbage value with addition
        c[i] = a[i] + b[i];
        
        // Printing the result immediately
        printf("Third array element at index %d is: %d\n", i, c[i]);
    }

    return 0;
}
