Topic: Concatenating Two Strings (In-Depth)

🔗 What is String Concatenation?

Concatenation means taking one string (the Source) and appending it to the end of another string (the Destination).

Goal: We want to take S2 ("khatri") and paste it at the end of S1 ("Jenny") so the final output becomes "Jennykhatri".

🧠 Visualizing Strings in Memory

Before writing code, look at how the compiler stores these strings based on the instructor's whiteboard drawing.

*Notice the Null Character \0 at index 5. This marks the end of S1.

Source (S2) : char S2[7] = "khatri";

To concatenate, we must overwrite S1's \0 (at index 5) with the first character of S2 ('k'), and continue copying until S2's \0 is copied.

⚠️ The Golden Rule: Array Size

Look at S1 above. It only has 6 blocks of memory. If we try to shove "khatri" into it, where does it go? It overwrites random memory next to the array!

Buffer Overflow: This happens when you try to put more data into an array than it was built to hold. It causes severe program crashes and security vulnerabilities.

The Fix: The Destination array MUST be declared large enough to hold both strings plus the final \0.

// WRONG
char S1[6] = "Jenny"; 

// CORRECT (Make S1 large enough, e.g., 30)
char S1[30] = "Jenny";

🛠️ Method 1: Using built-in `strcat()`

C provides a library function to do this automatically.

  • Header required: #include <string.h>
  • Syntax: strcat(destination, source);
  • Return value: It returns a pointer to the destination string.
char S1[30] = "Jenny";
char S2[] = "khatri";

// S2 is appended to S1
strcat(S1, S2); 

printf("%s", S1); // Output: Jennykhatri

The `strncat` variant

If you only want to append a specific number of characters, use strncat.

strncat(S1, S2, 4); // Appends only the first 4 chars ("khat")

⚙️ Method 2: Custom Logic (Without strcat)

To truly understand C, we must write the logic manually using a for loop.

The Algorithm:

  1. Find the length of S1 (Where does S1 end? Index 5).
  2. Find the length of S2 (How many characters do we copy? 6).
  3. Loop through S2 and place characters into S1, starting at S1's end.
int len1 = strlen(S1); // len1 = 5
int len2 = strlen(S2); // len2 = 6

// Important: Loop runs <= len2 so the \0 is copied!
for(int i = 0; i <= len2; i++) {
    S1[len1 + i] = S2[i];
}

Dry Run Visualization

Let's step through the loop to see exactly how memory changes.

Step (i) Action: S1[5 + i] = S2[i] Result in S1
i = 0 S1[5] = S2[0] ('k') Overwrites '\0' with 'k'
i = 1 S1[6] = S2[1] ('h') Adds 'h'
i = 2 S1[7] = S2[2] ('a') Adds 'a'
... ... ...
i = 6 S1[11] = S2[6] ('\0') Safely closes the new string with '\0'

💻 Full Practical Code

Here is the complete program written by the instructor in VS Code, combining everything discussed.

#include <stdio.h>
#include <string.h>

int main() {
    // 1. Declaration (Ensure S1 has enough space!)
    char S1[30] = "Jenny";
    char S2[] = "khatri";
    
    int len1, len2, i;
    
    // 2. Calculate Lengths
    len1 = strlen(S1);
    len2 = strlen(S2);
    
    // 3. Custom Concatenation Loop
    for(i = 0; i <= len2; i++) {
        S1[len1 + i] = S2[i];
    }
    
    // Alternative: built-in function (Comment out the loop to test)
    // strcat(S1, S2);
    
    // 4. Print Result
    printf("String after concatenation is: %s\n", S1);
    
    return 0;
}
