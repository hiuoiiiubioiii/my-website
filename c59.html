📐 1. The Mathematical Rule

Before writing code, we must understand how matrix addition works on paper. The instructor notes one Golden Rule:

You can ONLY add matrices if they have the exact same dimensions.

If Matrix A is 2x3 (2 rows, 3 columns), Matrix B must also be 2x3. The result, Matrix C, will also be 2x3. Addition is performed element-by-element.

Matrix A [2][3]
123 101
+
Matrix B [2][3]
111 111
=
Matrix C [2][3]
234 212

💾 2. Memory & Declarations

To store these matrices in C, we need 2D Arrays. A 2D array is declared with the number of rows first, then the columns.

int a[2][3], b[2][3], c[2][3]; // 3 matrices
int i, j;                      // Loop counters

Here, i will control the rows, and j will control the columns.

📥 3. Reading Inputs (Nested Loops)

To populate a 2D array, we use nested `for` loops. The outer loop runs for every row, and the inner loop iterates through every column of that specific row.

// Reading Matrix A
for(i = 0; i < 2; i++) {       // Row loop (0 to 1)
    for(j = 0; j < 3; j++) {   // Column loop (0 to 2)
        scanf("%d", &a[i][j]); 
    }
}

Note: The instructor writes this exact same block twice on the whiteboard—once for &a[i][j] and once for &b[i][j].

⚙️ 4. The Addition & Printing Logic

Efficiency Trick: You do not need a separate set of loops just to print the result. As taught on the whiteboard, we calculate the sum and print it inside the same loop.

for(i = 0; i < 2; i++) {
    for(j = 0; j < 3; j++) {
        // 1. Add matching elements
        c[i][j] = a[i][j] + b[i][j]; 
        
        // 2. Print immediately with a tab
        printf("%d\t", c[i][j]); 
    }
    // 3. Newline after a row finishes
    printf("\n"); 
}

Why use \t and \n?

  • \t (Tab): Pushes the next number to the right, keeping elements aligned in neat columns.
  • \n (Newline): Pushes the cursor to the next line so the next row prints underneath, creating the visual grid.

Step 3: i = 0

Inner loop increments. j = 2 (Column 2)

c[0][2] = a[0][2] + b[0][2] ➔ -1 + 0 = -1

Prints: -1 [tab]

Now j = 3. The condition j < 3 fails. The inner loop exits. The program executes printf("\n"); moving the cursor to the next line. Then i becomes 1, and the process repeats for Row 1.

💻 6. Full VS Code Implementation

Here is the final, polished code the instructor writes in the IDE. She adds standard print statements to make it user-friendly, as well as logic to print out the original matrices before the sum.

#include <stdio.h>

int main() {
    int a[2][3], b[2][3], c[2][3], i, j;

    // --- STEP 1: READ FIRST MATRIX ---
    printf("Enter first matrix:\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            scanf("%d", &a[i][j]);
        }
    }

    // --- STEP 2: READ SECOND MATRIX ---
    printf("Enter second matrix:\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            scanf("%d", &b[i][j]);
        }
    }

    // --- STEP 3: PRINT FIRST MATRIX ---
    printf("\nThe first matrix is:\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            printf("%d\t", a[i][j]);
        }
        printf("\n");
    }

    // --- STEP 4: PRINT SECOND MATRIX ---
    printf("\nThe second matrix is:\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            printf("%d\t", b[i][j]);
        }
        printf("\n");
    }

    // --- STEP 5: CALCULATE AND PRINT THE SUM ---
    printf("\nThe Third matrix (Sum) is:\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            c[i][j] = a[i][j] + b[i][j]; 
            printf("%d\t", c[i][j]);     
        }
        printf("\n");                    
    }

    return 0;
}

📝 7. Homework Assignment

At the end of the video, the instructor provided a challenge: Write a C program to perform the subtraction of two matrices.

Instructor's Hint: The logic is virtually identical! Because matrix subtraction also happens element-by-element, you just need to change your inner logic to:

c[i][j] = a[i][j] - b[i][j];

Try writing this in your IDE and testing it with negative numbers!
