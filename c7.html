1. Introduction & Definition

Before writing programs, you must understand the basic building blocks: Variables, Keywords, and Constants.

Definition: Constants are entities with fixed values. These values cannot be changed during the execution of the program.

Symbolic Constants

Defined in the "Definition Section" of the C program. Whenever you use the name, the compiler replaces it with the fixed value before compiling.

#define PI 3.14

Throughout the program, wherever you use PI, the value 3.14 will be used. You cannot change this value later in the code.

3. Numeric Constants: Integers

Integer constants are whole numbers without decimal points. They can be represented in three different number systems in C.

The 3 Number Systems

  • Decimal (Base 10): Uses 0-9. Cannot start with 0 (unless the number itself is just 0).
  • Octal (Base 8): Uses 0-7. Rule: Must start with a 0.
  • Hexadecimal (Base 16): Uses 0-9 and A-F. Rule: Must start with 0x or 0X.

✅ Valid Integers

10, 5, 0 -123, +1234 05 (Valid Octal) 0X74, 0xAF (Valid Hex)

❌ Invalid Integers

08 (Invalid Octal: 8 is not in base 8) 0x7G (Invalid Hex: G is not valid) 56,100 (Commas not allowed) 57 100 (Spaces not allowed) $120 (Special chars not allowed)

4. Numeric Constants: Real (Floating)

Numbers containing a decimal point (fractional parts). Also known as floating-point constants.

✅ Valid Real Constants

12.56 +12.56 -56.02

❌ Invalid Real Constants

12.56.02 (Cannot have two decimal points)

Note: Real constants can also be represented in Exponential form (e.g., Mantissa e Exponent).

5. Character Constants

This category is split into Single Characters and Strings. It is crucial to understand how they are stored in memory.

A. Single Character Constants

A single character enclosed in Single Quotation Marks ' '.

  • Examples: 'a', '5', ' ' (Space), '$', '\0' (Null).
  • Internal Storage: Computers don't store characters; they store integers known as ASCII Codes.

    'A' = 65  |  'a' = 97  |  '0' = 48
CRITICAL DISTINCTION: 5 vs '5'
5 is an Integer (Value is exactly 5).
'5' is a Character (The computer sees the ASCII value 53).

If you write printf("%d", 'a');, it will print the integer 97.

B. String Constants

A sequence of characters enclosed in Double Quotation Marks " ".

  • Examples: "Jenny", "abc", "12345", "a".

How Strings are stored in Memory

The compiler automatically adds a Null Character \0 at the end of every string.

String: "Jenny" (Looks like 5 characters)
J1
e2
n3
n4
y5
\06

Actual Memory Length: 6 Bytes (5 characters + 1 Null Terminator)

CRITICAL DISTINCTION: 'a' vs "a"
'a' is a Single Character (Length: 1 Byte).
"a" is a String Constant (Contains 'a' + '\0', Length: 2 Bytes). They are NOT equal!

6. How to Declare Constants in Code

There are two primary ways to declare constants in a C program.

Method 1: Using the const Keyword

You must initialize the variable at the exact time of declaration. Once declared, it becomes "Read Only".

#include <stdio.h>

void main() {
    const int a = 10;   // Correct declaration
    
    // a = 50;          // ❌ ERROR: Assignment of read-only variable
    
    printf("%d", a);
}

Method 2: Using the #define Directive

This is a preprocessor directive, meaning the compiler replaces the symbolic name with the value before actual compilation begins.

Rules for #define

  • Usually written above main().
  • No Semicolon (;) at the end.
  • No Assignment Operator (=).
  • Convention is to use Uppercase letters.
#include <stdio.h>
#define MAX 50          // ✅ Correct

// #define MAX = 50;    // ❌ Incorrect

void main() {
    printf("Max value is %d", MAX);
}

Lecture Summary

Constants represent fixed values. Integers can be decimal, octal (starts with 0), or hex (starts with 0x). Characters are stored as ASCII integers and use single quotes, while Strings use double quotes and are secretly terminated by a null character (\0). You can enforce constancy in code using the const keyword or the #define macro.
