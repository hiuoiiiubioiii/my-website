Formatted Output in C

Mastering the printf() Function | Visual Study Notes

1. Introduction to Output in C

In C programming, input and output statements can be classified into two types:

  • Unformatted Functions: Deal with raw characters and strings (to be discussed later).
  • Formatted Functions: Allow you to arrange the output in a specific layout, format, and structure.

When we talk about Standard Output, we are referring to the default output device, which is your monitor/screen. (Standard Input refers to the keyboard).

Key Takeaway: printf() is the standard formatted output function in C. It prints data directly to your monitor in exactly the format you specify.

2. General Syntax: printf() vs scanf()

The syntax for printf() looks very similar to scanf(), but there is one massive difference regarding memory allocation.

printf("Control string", arg1, arg2, ... argn);
  • Control String: Enclosed in double quotes " ". It can contain plain text, format specifiers (like %d, %f), and escape sequences (like \n).
  • Arguments: The variables whose values you want to print.
🚨 The Golden Rule: NO Ampersands (&)
In scanf(), you write &a because you are passing the memory address of the variable to store keyboard input.

In printf(), you simply write a. You don't need the address; you are passing the actual value (Pass by Value) stored inside the variable to be printed on the screen.

3. Memory Visualization: How it Executes

int a, b, sum; printf("Enter two numbers: "); scanf("%d %d", &a, &b); // User enters 5 and 4 sum = a + b; printf("Sum = %d", sum);

When the compiler hits printf("Sum = %d", sum);, it sees the %d format specifier. It looks at the first argument after the comma (sum). It goes to memory, grabs the value 9 from address 3000, and replaces the %d with that 9 on the screen.

4. Printing Multiple Variables & Text

You can mix plain text and multiple format specifiers freely. The compiler maps the specifiers to the arguments in sequential order.

printf("a = %d, b = %d, sum = %d", a, b, sum);

Screen Output:

a = 5, b = 4, sum = 9

Using Escape Sequences

To print outputs on different lines, use the newline character \n inside the control string.

printf("a = %d \n b = %d \n sum = %d", a, b, sum);

Screen Output:

a = 5
b = 4
sum = 9

5. Advanced Integer Formatting (Width & Justification)

This is where printf gets powerful. You can specify exact field widths to make your output align perfectly like a table or a receipt.

Let's declare an integer: int a = 1234;

Right Justified (Default)

printf("%6d", a);

The 6 tells the compiler to reserve a minimum of 6 character blocks. Since 1234 is only 4 digits, it leaves 2 blank spaces on the left. By default, output pushes to the right.

Left Justified

printf("%-6d", a);

Adding a minus sign (-) forces the text to align to the left side of the reserved blocks. The blank spaces are placed on the right.

Zero Padding

printf("%06d", a);

Putting a 0 before the width number tells the compiler to fill any empty left-side space with zeros instead of blanks.

6. Float Precision Formatting

By default, the %f specifier prints up to 6 decimal places. We use the syntax %[width].[precision]f to control it.

Let's declare a float: float a = 1234.3456;

printf("%10.2f", a);
  • .2 (Precision): Print exactly 2 numbers after the decimal. The compiler rounds the number. .345 rounds up to .35.
  • 10 (Total Width): Reserve exactly 10 blocks. 1234.35 takes up 7 blocks (the decimal point counts as a block!). This leaves 3 blank spaces on the left.

Other Variable Types:
You can apply these exact same width rules to characters using %c (e.g., %5c will print a right-justified character in a 5-block space).

You can also print scientific/exponential numbers using the %e format specifier!
