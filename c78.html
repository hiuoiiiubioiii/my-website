🖨️ 2. Basic Dereferencing

    printf("%d\n", *p);

The * (indirection/dereference operator) goes to the address stored in p (which is 100) and pulls out the value.

⚠️ 3. The "Tricky" Triple Print (Crucial Interview Concept)

This is the most important line in the lecture. Notice that there are three operations separated by commas inside the printf function.

    printf("%d %d %d\n", (*p)++, *p++, *++p);
The Golden Rule: Function arguments in C (like in GCC compilers) are evaluated from Right-to-Left, but printed Left-to-Right. Let's break down the execution right to left!
Step 1
(Right)
Evaluate *++p: Pre-increment the pointer first. p moves from 100 to 104. Then dereference it. The value at 104 is 11. (Passed to printf: 11)
Step 2
(Middle)
Evaluate *p++: Post-increment the pointer. Dereference the current address first. p is at 104, so value is 11. After returning 11, p increments to 108. (Passed to printf: 11)
Step 3
(Left)
Evaluate (*p)++: Post-increment the value. p is at 108. The value is -1. It returns -1 to printf, and then increments the actual array value in memory by 1 (-1 + 1 = 0). The array is permanently modified! (Passed to printf: -1)

Because the print matches the %d placeholders left-to-right, the final Output is: -1 11 11

➡️ 4. Pointer Arithmetic (Setting up Q)

    q = p + 3;
    printf("%d\n", *q - 3);

Logic: p is at address 108. Adding 3 means jumping forward 3 integers (12 bytes). 108 + 12 = 120. q now points to address 120.

For the print statement: Dereference q. The value at 120 is 5. Then subtract 3. (5 - 3 = 2).

Output: 2

🔙 5. Pre-Decrementing a Pointer

    printf("%d\n", *--p + 5);

Logic: The unary operators * and -- have higher precedence than +. They evaluate right-to-left.

  1. --p: Pre-decrement moves p back one space from 108 to 104.
  2. *p: Dereference value at 104, which is 11.
  3. + 5: Add 5. (11 + 5 = 16).

Output: 16

➕ 6. Final Addition

    printf("%d\n", *p + *q);
} // End of main program

Logic: Simply dereference both pointers based on their final positions.

  • *p is at 104, value is 11.
  • *q is at 120, value is 5.
  • 11 + 5 = 16.

Output: 16

💡 Instructor's Hints for Success:

  1. q-- permanently moves the pointer back 4 bytes.
  2. *(q+2) does not permanently move q. It just calculates a temporary address, looks ahead 2 spaces, and grabs the value.
  3. Watch out for p++ inside parentheses! Remember associativity rules and that postfix means it evaluates the expression first, and moves p afterward.
