Append Mode in C 📂

Scroll to learn how to add data to existing files without losing content.

1. The Core Concept

Before this lesson, we discussed two primary file handling modes:

  • Read Mode ("r"): Look at the content only. No modifications allowed.
  • Write Mode ("w"): Write data to a file, but it erases (overwrites) everything previously inside it.

Append Mode ("a") is the solution when you want to keep the existing data and simply add new data to the very end of the file.

3. Step-by-Step Code Implementation

Here is the complete C program the instructor writes, combining her whiteboard code with her verbal explanations.

append_file.c 
#include <stdio.h>
#include <stdlib.h>

void main() {
    // 1. Create a file pointer & initialize to NULL
    FILE *fp = NULL; 
    
    // 2. Create a buffer to hold the user's input
    char str[50]; 

    // 3. Open the file in APPEND mode ("a")
    fp = fopen("abc.txt", "a"); 

    // 4. Error Handling (If file fails to open)
    if (fp == NULL) {
        printf("error");
        exit(1);
    }

    // 5. Ask user what they want to add
    printf("Enter the content you want to append: ");
    
    // 6. Read input (gets reads strings with spaces)
    gets(str); 

    // 7. Write the string into the file
    // Notice the \n adds the new text to a new line
    fprintf(fp, "\n%s", str);  
    
    // 8. Success message to console
    printf("Successfully appended");

    // 9. Save memory and prevent data corruption
    fclose(fp);
}

4. Functions Used to Append

Once a file is opened in Append Mode ("a"), you can use any standard writing function. The instructor highlights three options:

fprintf(fp, "%s", str);

Best if you want to control formatting. You can easily add \n here to push appended data to the next line.

fputs(str, fp);

A simple function specifically designed for putting a standard string directly into a file.

fputc(char, fp);

Used to append a single character. If you want to write a whole string using this, you must place it inside a while or for loop.

5. Exam Prep: "w" vs "a" Mode

This is a highly tested concept in exams and interviews. Understand the exact differences:

Scenario Write Mode ("w") Append Mode ("a")
If file DOES NOT exist Creates a brand new file. Creates a brand new file.
If file DOES exist Erases previous content. Keeps previous content.
Cursor Location Starts at the beginning. Starts at the very end.
