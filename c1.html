💡 Why Do We Need Programming?

A computer is a General Purpose Machine designed to perform computational tasks. However, it has a major limitation: Communication.

Human: "Hey Computer, add 4 and 5!"

⬇️

Computer: "❌ I don't understand English."

To solve this, we must provide a Sequence of Instructions. These sets of instructions are called Programs.

📂 Types of Programs (Software)

Software is simply a collection of programs. They fall into two main categories:

  • 1. System Software: Manages system resources and hardware (e.g., Operating Systems like Windows, Linux).
  • 2. Application Software: Performs specific tasks for the user (e.g., Web Browsers, MS Office, Text Editors).

Note: If you need a specific task done that isn't covered by existing software, you must write your own custom program.

🤖 How Computers Think (Machine Language)

Computers only understand Binary (0 and 1). This is called Machine Level Language. We cannot write English; we must write streams of bits.

Example: Adding 1 + 2

The instructor visualizes an "Instruction Register" on the board showing how a CPU sees this math problem:

Instruction Format (20 Bits example)
Opcode (Operation) 0010
Operand 1 (The number 1) 00000001
Operand 2 (The number 2) 00000010
⬇️ Sent to CPU

Here, 0010 is a code meaning "PLUS". This string of numbers is extremely difficult for humans to memorize and write.

⚠️ The Problem: Machine Dependency

Writing in binary (Machine Language) has a huge flaw: Non-Portability.

CPU Architecture A:
Plus = 0010

CPU Architecture B:
Plus = 1101

A program written for Computer A will not work on Computer B because their internal wiring (Architecture) is different. You would have to rewrite the entire code for every new machine.

📜 Enter C Language

To solve the difficulty of Machine Language, High-Level languages were created. C is one of the most important ones.

Key Facts:

  • Developer: Dennis Ritchie
  • Year: 1972
  • Location: Bell Laboratories (AT&T), USA
  • Predecessor: B Language (developed by Ken Thompson)
  • Purpose: C was created specifically to write the UNIX Operating System.

Note: In 1989, the language was standardized by ANSI (American National Standards Institute) to ensure code worked the same everywhere.

⚙️ How C Works: The Compiler

C is a High-Level Language. We use English words like printf, main, and include. However, the CPU still only understands binary.

We use a translator program called a Compiler to bridge this gap.

The Execution Flow
Source Code (.c)
(Human Readable: printf, main)
COMPILER
Object Code / Machine Code
(0s and 1s)

This makes C Portable. You write the code once, and the compiler translates it for whatever machine you are using.
