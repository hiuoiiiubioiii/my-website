The Core Rule

You can ONLY multiply two matrices if the Columns of Matrix A == Rows of Matrix B.

  • In our example: n (3) == p (3). Multiplication is possible.
  • The resulting Matrix C will take the remaining dimensions: Rows of A × Columns of B (3 × 2).

2. Building the Core Logic (Nested Loops)

To multiply these matrices, we need three nested loops. Here is the logic the instructor writes on the whiteboard.

for(int i = 0; i < 3; i++)          // Loop 1: Tracks ROWS of Matrix C
{
    for(int j = 0; j < 2; j++)      // Loop 2: Tracks COLUMNS of Matrix C
    {
        sum = 0;                // CRUCIAL: Reset sum for every new cell
        
        for(int k = 0; k < 3; k++)  // Loop 3: Handles the moving math pointer
        {
            sum = sum + a[i][k] * b[k][j]; 
        }
        c[i][j] = sum;          // Store final sum in Result Matrix C
    }
}

Why 3 Loops?

  • The i loop: Determines which row of Matrix A we are using.
  • The j loop: Determines which column of Matrix B we are using.
  • The k loop: Moves across the columns of Matrix A and down the rows of Matrix B simultaneously to multiply and add the items.

5. Practical C Code Implementation

Moving from the whiteboard to the IDE. Here are the crucial additions made in the final C code:

  • Dynamic Sizing: Used #define N 50 so matrices can be any size up to 50x50.
  • The Safety Check: You must use an if (n != p) check before starting the loops to prevent mathematical errors.
#include <stdio.h>
#define N 50

int main() {
    int a[N][N], b[N][N], c[N][N];
    int i, j, k, sum;
    int m, n, p, q;

    // 1. Ask for dimensions of Matrix A
    printf("Enter rows and columns for first matrix:\n");
    scanf("%d %d", &m, &n);

    // Read Matrix A data... (Code omitted for brevity)

    // 2. Ask for dimensions of Matrix B
    printf("Enter rows and columns for second matrix:\n");
    scanf("%d %d", &p, &q);

    // Read Matrix B data...

    // 3. THE SAFETY CHECK
    if (n != p) {
        printf("Can not multiply\n");
    } 
    else {
        // 4. THE MULTIPLICATION LOGIC
        for(i = 0; i < m; i++)          // Outer loop: rows of A
        {
            for(j = 0; j < q; j++)      // Inner loop: cols of B
            {
                sum = 0;
                for(k = 0; k < n; k++)  // Math loop: matching dimension (n or p)
                {
                    sum = sum + a[i][k] * b[k][j];
                }
                c[i][j] = sum;
            }
        }
        
        // Print Result Matrix C...
    }
    return 0;
}

Instructor's Note: Always remember that the k loop bounds must be the matching dimension (either n or p since they are equal). The resulting matrix loop bounds are m (rows of A) and q (cols of B).
