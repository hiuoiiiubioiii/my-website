Topic: Reversing a String in Memory (In-Depth Logic & Code)

1. What does "Reversing" actually mean?

The instructor emphasizes a crucial point: Reversing a string is not just running a loop backwards and using printf to display characters from the end to the beginning.

True reversal means modifying the actual array in memory by swapping the characters. Let's visualize the string "Jayanti".

Odd vs. Even Lengths: The logic we write must work for both. For odd lengths (like 7), the exact middle character doesn't move. For even lengths (like 8), every single character will find a swap partner.

2. Method 1: The Built-in Function

C provides a pre-defined function inside the <string.h> library to do this automatically.

#include <stdio.h>
#include <string.h>

void main() {
    char s1[30] = "Jayanti";
    strrev(s1); // Permanently reverses s1 in memory
    printf("%s", s1); // Outputs: itnayaJ
}
⚠️ Instructor's Warning on Compilers:
strrev is considered a non-standard function. While it works perfectly on older compilers like Turbo C, modern compilers (like GCC used in VS Code, Linux, or Mac) might throw an "undefined reference" error. Therefore, learning the custom logic is absolutely mandatory for exams and technical interviews.

3. Method 2: Custom Logic (Single Variable Loop)

To reverse the string manually, we need a loop. But how many times should it run?

If we loop through the entire length of the string, we will swap the characters, and then accidentally swap them right back to their original positions. Therefore, we only loop up to half the length of the string.

  • Length (l) = 7
  • l / 2 = 7 / 2 = 3 (Integer division drops decimals).
  • The loop runs 3 times (Indices 0, 1, and 2).

The Math Formula for Swapping:

If our left index is i, what is the right index? The formula is length - 1 - i.

  • If i=0: 7 - 1 - 0 = 6 (Swap 0 and 6)
  • If i=1: 7 - 1 - 1 = 5 (Swap 1 and 5)

The 3-Step Swap Logic:

// 'ch' is a temporary character variable
ch = s1[i];                // Step 1: Backup the left character
s1[i] = s1[l - 1 - i];     // Step 2: Overwrite left with right
s1[l - 1 - i] = ch;        // Step 3: Put backup into the right

4. Alternative Logic: The Two-Pointer Approach (Best Way)

Using the math formula l - 1 - i can be confusing to read. The instructor introduces a much cleaner approach using two variables inside the for loop.

  • Variable i starts at the beginning (0) and moves right (i++).
  • Variable j starts at the end (length - 1) and moves left (j--).
  • The loop stops when i and j meet or cross (i < j).

5. Final Practical Implementation

This is the exact, final code the instructor writes, compiles, and runs in VS Code using the Two-Pointer logic.

#include <stdio.h>
#include <string.h>

void main() {
    int l, i, j;
    char s1[30], c; // 'c' is our temporary variable

    printf("Enter string: ");
    gets(s1);

    // Find total length
    l = strlen(s1);

    // Reversing Loop (Two-Variable Approach)
    for(i = 0, j = l - 1; i < j; i++, j--) {
        
        // Swapping Logic
        c = s1[i];       
        s1[i] = s1[j];   
        s1[j] = c;       
    }

    printf("String is: %s", s1);
}
